130 /* From CiA402, - State coding
131  Statusword | PDS FSA state
132 xxxx xxxx x0xx 0000 | Not ready to switch on
133 xxxx xxxx x1xx 0000 | Switch on disabled
134 xxxx xxxx x01x 0001 | Ready to switch on
135 xxxx xxxx x01x 0011 | Switched on
136 xxxx xxxx x01x 0111 | Operation enabled
137 xxxx xxxx x00x 0111 | Quick stop active
138 xxxx xxxx x0xx 1111 | Fault reaction active

140 */
