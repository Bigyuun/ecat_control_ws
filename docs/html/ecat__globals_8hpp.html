kVelocityControlStructure 

0-3 , 4 bits. Val : 1 - PI current controller

4-7, 4bits. Val : 0 - None | 1 - PI Vecolity controller (low pass filter) | 2 - PI velocity controller (observer)

8-11 , 4bits. Val : 0 - None | 1 - PID position controller

1 bit Val : 0 - None | 1 - Gear Mounted on system
